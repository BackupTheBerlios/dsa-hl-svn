Beschreibung

	Ein simples Programm zum Umwandeln von HalfLife in Quake3Maps.

Anleitung

Map-Datei
	Die Umzuwandelnde Datei, eine korrekte HammerMap

Wad-Verzeichnis
	Das verzeichnis, das die Wad-Dateien, die in der Map benutzt werden, enthält.
	Weiterhin sucht der Converter in den relativen Verzeichnissen aus der Map
	auf allen Festplatten, sollte er eine Wad nicht im gegebenen Verzeichnis finden.
